# Single Page CV - HTML Semantik
https://roadmap.sh/projects/single-page-cv

Projek ini adalah implementasi resume profesional satu halaman menggunakan **HTML5 Semantik** murni. Fokus utama projek ini adalah pada struktur data yang bersih, aksesibilitas (accessibility), dan kemudahan dibaca oleh sistem ATS (Applicant Tracking System).

## 🚀 Fitur Utama
- **Semantik HTML5**: Menggunakan tag seperti `<header>`, `<main>`, `<section>`, `<article>`, dan `<address>`.
- **Aksesibilitas**: Struktur tajuk (heading) yang hierarkis untuk pembaca skrin (screen readers).
- **Layout Profesional**: Mengikuti template resume standard industri.
- **Pautan Dinamik**: Semua pautan sosial dibuka di tab baru dengan atribut keselamatan `rel="noopener noreferrer"`.

## 🛠️ Teknologi yang Digunakan
- HTML5
- Konsep Semantic Web
